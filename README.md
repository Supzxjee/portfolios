# Lê Đức Anh | Portfolio

Chào mừng bạn đến với kho lưu trữ cho trang web portfolio cá nhân của mình! Dự án này giới thiệu các kỹ năng, kinh nghiệm và các dự án của mình với tư cách là một  DevOps & AI Engineer.

## 🎨 Thiết kế & Giao diện
Trang portfolio được thiết kế với giao diện dark-mode mang phong cách terminal (cửa sổ dòng lệnh). Dự án sử dụng các điểm nhấn màu neon, các thành phần glassmorphism (hiệu ứng kính mờ) và hiệu ứng gõ phím mượt mà nhằm mang lại một không gian đậm chất lập trình viên.

## 🛠️ Công nghệ sử dụng
- **HTML5**: Xây dựng cấu trúc Semantic.
- **Tailwind CSS**: Framework CSS tiện ích giúp phát triển giao diện nhanh chóng.
- **Vanilla JavaScript**: Xử lý các tương tác như hiệu ứng gõ phím, menu trên thiết bị di động, và hiệu ứng cuộn mượt mà.
- **Lucide Icons**: Bộ icon hiện đại, đồng nhất.
- **Google Fonts**: Sử dụng font Inter (sans-serif) và JetBrains Mono (monospace).

## 🧩 Các phần chính của trang web
- **Hero/Home**: Phần giới thiệu với hiệu ứng gõ chữ kiểu terminal và các liên kết nhanh.
- **Tech Stack (Kỹ năng)**: Tổng quan về các kỹ năng được phân loại theo Infrastructure & Cloud (Hạ tầng & Đám mây), Languages (Ngôn ngữ lập trình), AI & Data (Trí tuệ nhân tạo & Dữ liệu), và Tools (Công cụ).
- **Projects (Dự án)**: Các dự án nổi bật bao gồm AI Observability Platform (Nền tảng giám sát AI), PCA-Guided CNN, và Algorithm Visualizer (Trực quan hóa thuật toán).
- **Experience & Achievements (Kinh nghiệm & Thành tựu)**: Các chứng chỉ và mốc thời gian phát triển bản thân.
- **Contact (Liên hệ)**: Biểu mẫu liên hệ tối giản cùng với các liên kết đến hồ sơ GitHub và LinkedIn của mình.

## 💻 Chạy dự án ở Local
Để chạy dự án này trên máy của bạn, chỉ cần clone kho lưu trữ về và mở file `index.html` trong trình duyệt web. Không cần cài đặt các công cụ build hay thư viện phức tạp.

```bash
# Clone kho lưu trữ
git clone https://github.com/Supzxjee/portfolios.git

# Di chuyển vào thư mục dự án
cd portfolios

# Mở file index.html (ví dụ: sử dụng Live Server trong VS Code hoặc chỉ cần click đúp vào file)
```
